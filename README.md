# 📊 Meu Dashboard

Este projeto consiste no desenvolvimento de um dashboard web criado do zero durante meus estudos em programação, com foco na prática de integração com APIs reais e autenticação segura.

A aplicação utiliza login com Google (OAuth 2.0) e consumo de API externa, explorando conceitos modernos de desenvolvimento com o apoio de Inteligência Artificial como ferramenta de aprendizado.
